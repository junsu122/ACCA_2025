# 🌸 붓꽃(Iris) 데이터를 이용한 신경망 구현

Scikit-learn의 붓꽃 데이터셋을 활용하여 입력층, 은닉층, 출력층을 가진 신경망을 학습시키고 꽃의 종류를 분류해 봅니다.

---

### 1️⃣ 데이터셋 구조
- **Feature (4개):** 꽃받침 길이/너비, 꽃잎 길이/너비
- **Target (3개):** Setosa, Versicolor, Virginica
- **학습 목표:** 4개의 특징을 입력받아 3개 중 어떤 품종인지 확률로 출력



---

### 2️⃣ 핵심 개념
- **활성화 함수 (ReLU):** 은닉층에서 비선형성을 추가하여 복잡한 데이터 패턴을 학습합니다.
    - 비선형의 특징을 추가하여 더 복잡한 예측을 가능하게 합니다.
- **Softmax:** 출력층에서 3개 클래스의 합이 1(100%)이 되도록 확률값으로 변환합니다.
- **역전파 (Backpropagation):** 출력값의 오차를 바탕으로 각 층의 가중치를 최적화합니다.



---
### 3️⃣ 모델 성능 확인 (Evaluation)
학습이 끝난 모델이 얼마나 정확한지 판단하기 위해 두 가지 방법을 사용합니다.

1. **Confusion Matrix (혼동 행렬)**
   - 어떤 클래스를 어떤 클래스로 착각했는지 표 형태로 보여줍니다.
   - 대각선 방향에 숫자가 몰려 있을수록 정확도가 높습니다.

2. **Classification Report**
   - **Precision (정밀도):** 모델이 정답이라고 예측한 것 중 진짜 정답의 비율
   - **Recall (재현율):** 실제 정답 중 모델이 맞힌 비율
   - **F1-score:** 정밀도와 재현율의 조화 평균
